Neo4j HA Agent

The Web UI bundle has not been built yet. Build it with:

cd ui && npm install && npm run build

Or run the agent's Maven build with the frontend module enabled. The compiled bundle will be served from this path (/) and the API is available at /api/*.

API endpoints currently exposed:

  • /health
  • /metrics
  • POST /api/login, GET /api/me
  • GET /api/cluster/status, POST /api/cluster/failover
  • GET /api/audit, GET /api/metrics-summary
